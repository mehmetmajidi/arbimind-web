## 🔧 تغییرات مورد نیاز در کد

### 1. MainChart Component
```typescript
// اضافه کردن Tab state
const [activeTab, setActiveTab] = useState<"chart" | "liquidation">("chart");

// اضافه کردن Tab UI
<Tabs>
  <Tab onClick={() => setActiveTab("chart")}>Chart</Tab>
  <Tab onClick={() => setActiveTab("liquidation")}>Liquidation Map</Tab>
</Tabs>

// Conditional rendering
{activeTab === "chart" && <CandlestickChart />}
{activeTab === "liquidation" && <LiquidationMap />}
```

### 2. Layout تغییرات
- اگر گزینه 2 را انتخاب کنیم: اضافه کردن به flex container زیر MainChart
- اگر گزینه 1 را انتخاب کنیم: تغییر در MainChart component
